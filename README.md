# 🌐 Cloned Frontend Projects Collection

This repository contains multiple **frontend clone projects** built using **HTML, CSS, and JavaScript**. These projects are created for learning, practice, and improving UI/UX development skills.

---

## 🚀 Projects Included

### 🪙 Bitcoin Magazine Clone

* A clone of the Bitcoin Magazine website
* Focus on layout, typography, and article design

### 📰 CNN Clone

* News website clone
* Includes homepage structure and news sections

### 🤖 OpenAI Clone

* UI inspired by OpenAI website
* Clean and modern landing page design

### 🔗 Polkadot Clone

* Blockchain-themed website clone
* Focus on animations and structured sections

### ❤️ Heart Up My Sleeve Clone

* Creative portfolio-style UI clone
* Emphasis on design and visuals

### 🍔 Zomato Clone

* Food delivery UI clone
* Includes restaurant listings and search UI

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## ⚙️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/frontend-clones.git
```

2. Open any project folder

3. Run the project

* Open `index.html` in your browser
